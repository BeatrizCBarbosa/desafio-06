Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • Parágrafo
  • Títulos
  • Imagens
  • Abreviações
  • Códigos-fonte
  • Listas ordenada
  • Listas desordenada
  • Quebra de linhas
  • Texto em itálico
  • Texto em negrito
  • Texto marcado

Atenção!

Não se esqueça de que sempre que abrir uma tag, você deve fecha-lá.
